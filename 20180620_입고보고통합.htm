Pharm IT3000 마약류통합관리시스템 연계

입고보고하기

 

◎ 준비사항

: 마약류통합관리시스템 연계 기능의 변경사항 적용을 위하여 Pharm IT3000 자동업데이트 진행 ( 환경설정 > 자동업데이트 )

 

◎ Pharm IT3000 마약류통합관리시스템 메뉴

: Pharm IT3000 실행 > 구매재고 > 마약류통합관리시스템 > 입고

 

◎ Pharm IT3000 마약류통합관리시스템 연계 기재고 입고보고

- 기재고 입고보고 : 5월 18일 이전 약국에서 사용하던 기재고에 대한 입고보고 방법입니다.

① 마약류통합관리시스템 > 기재고등록

: [마약류통합관리시스템] > [입고] > [신규보고]를 클릭하여 신규입고 선택 창이 뜨면 [기재고등록 바로가기] 클릭

② Pharm IT3000 기재고 자동 입고

: [Pharm IT3000 재고 불러오기] 클릭하여 Pharm IT3000 재고수량확인(낱알 총 수량)

 

③ 재고수량 수정

: 재고수량을 수정하려면 낱개단위 수량에 마우스 클릭 후 수정할 재고 입력 > ENTER 키를 누릅니다.

④ 재고약품 삭제

: 기재고등록할 약품 리스트에서 삭제하고자 하는 약품이 있을 경우

키보드 [delete]를 클릭하여 삭제

④ 기재고 등록 보고 완료

: [저장]을 클릭하여 기재고 등록 보고를 완료합니다.

(기재고 등록 작업을 2번 할 경우 재고가 중복 등록될 수 있으니, 수량변경시

 반드시 변경보고를 사용해주시기 바랍니다.)

 

 

◎ Pharm IT3000 마약류통합관리시스템 연계 기재고 추가 입고보고

기재고 등록 이후 5월 18일 이전 재고소진 또는 신규재고 입고 된 내역이 있을

경우 추가 기재고 입고보고를 진행합니다.

 

① 마약류통합관리시스템 > 기재고등록

: [마약류통합관리시스템] > [입고] > [신규보고]를 클릭하여 신규입고 선택 창이 뜨면 [기재고등록 바로가기] 클릭

② 약품명 또는 약품코드 입력

: 추가로 기재고 등록할 약품명 또는 약품코드를 제품명 또는 EDI코드 항목에 입력 > [제품검색] 클릭 > 해당 약품코드 선택

③ 재고수량 입력

: 낱알단위 수량에 등록할 재고수량 입력 > [입고] 클릭

④ 기재고 등록 보고 완료

: 입력한 약품 및 재고수량이 맞으면 [저장]을 클릭하여 기재고 등록 보고를 완료합니다.

 

◎ Pharm IT3000 마약류통합관리시스템 연계 자동입고 보고

5월 18일 이후 도매상에서 입고한 마약, 향정 약품에 대해 도매상에서 출고등록한

내역을 자동 입고보고 하는 메뉴입니다.

① 마약류통합관리시스템 > 자동입고

: [마약류통합관리시스템] > [입고] > [신규보고]를 클릭하여 신규입고 선택 창이 뜨면 [자동입고 바로가기] 클릭

② 입고대상내역 조회

: 신규입고 창이 뜨면 거래일시를 선택합니다. [입고대상내역조회] 클릭 시 선택한 거래 일시부터 도매상에서 출고 등록한 약품리스트를 조회 합니다.

 

③ 입고보고 대상내역 선택

: 입고보고 할 내역을 [선택] 한 후 [저장] 클릭합니다.

④ 저장소 선택

: 저장소정보입력 화면이 뜨면 보관할 저장소를 선택 후 [저장]을 클릭

하여 입고신규보고작업 완료합니다.

( 1개 이상의 입고대상내역을 선택했을 경우 선택한 입고대상내역 입고를 같은 저장소로 저장하려면 [일괄적용]에 체크 > [저장] 클릭)

⑤ 입고보고 내역 확인

: [입고] > [재고조회]를 클릭하면 입고 보고 내역을 확인할 수 있습니다. ( 마약류통합관리시스템홈페이지 > [보고관리] > [구입보고] 메뉴에서 입고보고 목록 확인 )

 

◎ Pharm IT3000 마약류통합관리시스템 연계 수동입고 보고

5월 18일 이후 도매상에서 사입한 마약, 향정 약품에 대해 수동 입고보고 하는 메뉴입니다.

약품 바코드, RFID 기기로 바코드 리딩이 가능한 경우 입력 방법입니다.

<RFID바코드>                           <바코드>

① 마약류통합관리시스템 > 수동

: [마약류통합관리시스템] > [입고] > [신규보고]를 클릭하여 신규입고 선택 창이 뜨면 [수동입고 바로가기] 클릭

② 신규 수동입고

: 저장소, 거래일시, 업체명, 입고단위, 거래처식별번호 등의 정보를 입력합니다.

③ 바코드 또는 RFID 정보 입력

: 바코드 또는 RFID 입력 란에 바코드, RFID를 통해 일련번호 입력 후 [입고]를 클릭합니다.

④ 입력한 정보가 그리드에 뜨면 [저장]을 클릭하여 입고보고를 완료합니다.

 

⑤ 입고보고 내역 확인

: [입고] > [재고조회]를 클릭하면 입고 보고 내역을 확인

( 마약류통합관리시스템홈페이지 > [보고관리] > [구입보고] 메뉴에서 입고보고 목록 확인 )

 

 

◎ Pharm IT3000 마약류통합관리시스템 연계 수동입고 보고 (수기입력)

5월 18일 이후 도매상에서 사입한 마약, 향정 약품에 대해 수동 입고보고 하는 메뉴입니다.

마약류 약품 포장에 필요한 일련번호 정보가 기재되어 확인이 가능한 경우에만 입력 가능합니다.

반드시 마약류 약품 포장에 (01)상품코드, (17)사용기한, (10)제조번호, (21)일련번호가 출력되어 있어야 수기입력 가능합니다.

 

<수기입력 가능 약품>                   <수기입력 불가능 약품>

① 마약류통합관리시스템 > 수동

: [마약류통합관리시스템] > [입고] > [신규보고]를 클릭하여 신규입고 선택 창이 뜨면 [수동입고 바로가기] 클릭

② 신규 수동입고

: 저장소, 거래일시, 업체명, 입고단위, 거래처식별번호 등의 정보를 입력합니다.

③ 바코드 또는 RFID 정보 입력방법

: 제품의 일련번호를 확인하여 입력 후 [입고]를 클릭합니다.

- 마약류 약품 포장에 (01)상품코드, (17)사용기한, (10)제조번호, (21)일련번호 순으로 입력합니다.

- 사용기한 입력은 6자리만 YYMMDD

- 입력시 구분 값(괄호 안의 숫자)도 같이 입력되어야 합니다.

(일련번호 수기입력 예시)

<약품1>                                <약품2>

<약품1> 0108806418060627171901101016005821170207J0106154

<약품2> 01088065000147291719080310AV0091212S5RWCCP4Z

(붉은색 표시 숫자는 구분값이고, 상품코드는 088로 시작됩니다.

구분값이 따로 기재되어 있지 않은경우 각 구분값 번호를 추가하여 같이 입력해주셔야 합니다.

상품코드(표준코드) 앞에 숫자’0’이 빠진경우 ‘0’을 추가하여 입력하셔야 합니다.)

 

④ 입력한 정보가 그리드에 뜨면 [저장]을 클릭하여 입고보고를 완료합니다.

 

⑤ 입고보고 내역 확인

: [입고] > [재고조회]를 클릭하면 입고 보고 내역을 확인

( 마약류통합관리시스템홈페이지 > [보고관리] > [구입보고] 메뉴에서 입고보고 목록 확인 )

 

◎ Pharm IT3000 마약류통합관리시스템 연계 입고 변경보고

- 입고 변경보고 : 기재고 등록 보고한 약품에 대한 수량 변경, 일부 약품 삭제 등의 변경사항이 있는 경우 변경보고 진행

1) 약품수량 변경 보고방법

① 마약류통합관리시스템 > [입고] > [변경보고] 클릭하여 변경보고 목록 조회

② 변경 보고 할 자료 선택 > 낱개수량 수정 > Enter키를 누릅니다.

③ 취소 또는 변경사유 입력 후 [저장]을 클릭합니다.

③ 마약류통합관리시스템홈페이지 > [보고관리] > [구입보고] 메뉴에서 변경보고 내역 확인

 

2) 일부 약품삭제 변경 보고방법

① 마약류통합관리시스템 > [입고] > [변경보고] 클릭하여 변경보고 목록 조회

② 변경보고 할 보고내역 선택 > 삭제시킬 약품의 삭제 항목에 체크 후 [저장]을 클릭합니다.

③ 취소 또는 변경사유 입력 후 [저장]을 클릭합니다.

③ 마약류통합관리시스템홈페이지 > [보고관리] > [구입보고] 메뉴에서 변경보고 내역을 확인합니다.

 

 

◎ Pharm IT3000 마약류통합관리시스템 연계 입고 취소보고

- 취소보고 : : 사용하지 않는 약품을 잘못 보고 한 경우 취소보고를 진행합니다.

(변경보고 대상인데 취소보고를 할 경우 식약처 집중 관리대상이 될 수 있으니 유의하시기 바랍니다.)

① 마약류통합관리시스템 > [입고] > [취소보고] 클릭하여 변경보고 할 목록 조회

② 취소 보고 할 자료 선택 > [저장] > 취소보고 사유 입력 후 > [저장] 클릭하여 취소보고 작업을 완료합니다.

③ 마약류통합관리시스템홈페이지 > [보고관리] > [구입보고] 메뉴에서 취소보고 내역 확인
